# Jira standup

cli tool to quickly show tickets you're assigned to and its impedements.

## Getting Started

### Installing

```
npm i -g jira-standup
```

### Configuration

Create a config file `.jirarc` to define your Jira url and username

```
username=johndoe
```

### Usage

```
jira standup
```
